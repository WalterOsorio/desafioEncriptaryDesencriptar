LEEME
Bienvenido a nuestro sitio web de Encriptador de Palabras.

Este sitio web está desarrollado utilizando JavaScript, un lenguaje de programación versátil y ampliamente utilizado para crear aplicaciones web interactivas y dinámicas. Nuestro encriptador de palabras permite tanto la encriptación como la desencriptación de palabras, brindando una forma sencilla y segura de proteger y revelar información textual.

Funcionalidades Principales:
Encriptación de Palabras: Introduce cualquier palabra en el campo de texto y nuestro algoritmo la convertirá en un formato cifrado. Este proceso asegura que la palabra original quede protegida y no sea fácilmente legible por terceros.
Desencriptación de Palabras: Si tienes una palabra previamente encriptada utilizando nuestra herramienta, simplemente insértala en el campo correspondiente y obtendrás la versión original y legible de la palabra.
Cómo Utilizar:
Encriptar:

Ingresa la palabra que deseas encriptar en el campo designado.
Haz clic en el botón "Encriptar".
La palabra encriptada se mostrará en pantalla y podrás copiarla para su uso posterior.
Desencriptar:

Ingresa la palabra encriptada en el campo correspondiente.
Haz clic en el botón "Desencriptar".
La palabra original aparecerá en pantalla.
Sobre el Proyecto:
Este encriptador de palabras ha sido diseñado con el objetivo de proporcionar una herramienta útil y accesible para aquellos que necesiten proteger sus palabras o mensajes de manera rápida y eficiente. Utilizando la flexibilidad y potencia de JavaScript, hemos creado una aplicación web que es fácil de usar y está disponible en cualquier dispositivo con acceso a internet.
